All of the Tor Browser Bundles have been updated to the latest Firefox 17.0.5esr.

**Tor Browser Bundle (2.3.25-6)**

- Update Firefox to 17.0.5esr
- Update NoScript to 2.6.59

**Tor Browser Bundle (2.4.11-alpha-2)**

- Update Firefox to 17.0.5esr
- Update NoScript to 2.6.59
